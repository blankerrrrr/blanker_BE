> 강조 표시된 내용이 MVP

## 0. 인증
- **로그인**
- **회원가입**

## 1. 필터 프로필 관리

- **관심 작품·인물·주제**
- 현재 진도 설정
- **차단할 정보 유형과 민감도 설정**
- 여러 프로필 생성 및 전환

## 2. 웹 콘텐츠 실시간 감지

- **웹페이지의 텍스트·이미지 분석**
- **스포일러·유해정보·관심정보 분류**
- **새롭게 로딩되는 콘텐츠 지속 감지**
- **콘텐츠 위험도와 관련도 판단**

## 3. 콘텐츠 차단 및 표시

- **위험 콘텐츠 자동 블러 처리**
- **텍스트·이미지·영역 단위 차단**
- **차단 사유와 관련 주제 표시**
- 콘텐츠별 일시 공개 또는 영구 허용

## 4. 나중에 보기 보관함

- **차단된 콘텐츠 저장**
- **원본 링크와 발견 위치 기록**
- 작품·주제·유형별 자동 분류
- 진도 달성 시 자동 잠금 해제
- 저장 콘텐츠 검색·삭제·읽음 처리

## 5. 놓친 정보 수집

- **관심 주제 관련 정보 자동 수집**
- 일정·공지·발매 정보 추출
- 스포일러 정보와 일반 정보 분리
- **중복 콘텐츠 통합**
- 일간·주간 요약 제공

## 6. 사용자 피드백 학습

- 오탐·누락 신고
- 특정 표현 허용 또는 차단
- 사용자 행동을 반영한 필터 조정
- 개인 맞춤형 탐지 정확도 개선

## 7. 자녀 보호

- 연령별 보호 프로필 설정
- 폭력·도박·혐오·사기·위험 행동 콘텐츠 차단
- 개인정보·금전·외부 연락 요구 감지
- 차단 이유와 안전 행동 안내
- 보호자 승인 요청 및 접근 기록 확인

## 8. 개인정보 및 설정

- 민감한 입력 영역 분석 제외
- 저장 데이터 자동 삭제
- 사이트별 예외 설정
- 기기 간 설정 동기화
